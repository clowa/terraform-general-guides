This repo is kind of my bulletin board. A collection of all my notes I made on my journey to learn terraform.

> [!TIP]
> Marp presentation files with the `.marp.md` extension are auto-rendered to PDFs on every push to main. You can download the latest PDFs by clicking the badge above.

## Structure

- `best_practices/`: Best practices for writing terraform code in general, not specific to modules.
- `introduction/`: General introduction to terraform and infrastructure as code.
- `modules/`: Best practices for writing terraform modules, testing terraform code, and deploying terraform code.
